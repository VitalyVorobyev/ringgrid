# DEC-006: Codebook Invariants

**Status:** Active
**Date:** 2025

## Decision

The marker codebook is a compile-time constant with strict mathematical
properties that must be preserved across regeneration.

### Properties (enforced by tests in `marker/codec.rs`)

Common to all shipped profiles:

1. **16-bit sectors** (`CODEBOOK_BITS = 16`).
2. **No rotational symmetry:** no codeword equals any of its 15 non-trivial
   cyclic rotations.
3. **Pairwise rotational uniqueness:** no two distinct codewords share any
   cyclic rotation.

Baseline profile (`CodebookProfile::Base`):

1. **893 codewords** (`CODEBOOK_N = 893`).
2. **Stable IDs `0..892`:** these codewords and their ordering are the shipped
   default contract and must not change silently.
3. **Minimum cyclic Hamming distance = 2**
   (`CODEBOOK_MIN_CYCLIC_DIST = 2`) between all pairs under all rotations.

Extended profile (`CodebookProfile::Extended`):

1. **2180 codewords** (`CODEBOOK_EXTENDED_N = 2180`).
2. **Additive prefix contract:** entries `0..892` are exactly the baseline
   profile in the same order; appended IDs start at `893`.
3. **Minimum cyclic Hamming distance = 1**
   (`CODEBOOK_EXTENDED_MIN_CYCLIC_DIST = 1`).
4. **No new polarity-complement collisions beyond the baseline:** appended
   entries exclude complement-equivalent duplicates relative to the full
   shipped profile, so enabling `extended` does not widen the baseline
   profile's existing polarity ambiguity.

### Decoding contract

- Matching: brute-force all `profile.len() × 16` rotations, pick lowest
  Hamming distance.
- Confidence: `clamp(1 − dist/6) × clamp(margin / profile.min_cyclic_dist())`
  where `margin = second_best_dist − best_dist`.
- Match result includes: `id` (codebook index), `rotation`, `dist`, `margin`,
  `confidence`.
- Default decode/config behavior uses `CodebookProfile::Base`; `extended`
  remains explicit opt-in.

### Generation

- Generated by `tools/gen_codebook.py` — never hand-edit `tools/codebook.json`
  or `crates/ringgrid/src/marker/codebook.rs`.
- Current committed baseline artifacts were generated with `--n 893 --seed 1`.
- The generator preserves the committed baseline profile as the fixed prefix,
  then appends the remaining rotationally unique 16-bit codewords whose
  complement classes are not already claimed by the shipped profile to build
  the opt-in extended profile.
- The achieved minimum distance written into each generated profile is the
  shipped truth for that profile:
  - baseline: `2`
  - extended: `1`
- Rebuild: `python3 tools/gen_codebook.py --n 893 --seed 1 ... && cargo build`.
